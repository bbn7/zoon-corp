Here's how to deploy your Next.js website manually to GitHub Pages:

Git repository hosted on GitHub
GitHub repository named: zoon-corp (for username: bbn7)
ðŸ”§ Manual Deployment Steps:
Step 1: Build Your Site

npm run build

This creates the out/ directory with static files.

Step 2: Create gh-pages Branch

# Create and switch to gh-pages branch
git checkout --orphan gh-pages

# Remove all files except those needed for the site
git rm -rf .
cp -r out/* .

# Add .nojekyll file to prevent Jekyll processing
touch .nojekyll

# Add all files
git add .
git commit -m "Deploy to GitHub Pages"


Step 3: Push to GitHub Pages Branch

# Push to gh-pages branch on GitHub
git push origin gh-pages
Step 4: Configure GitHub Pages
Go to your repository: https://github.com/bbn7/zoon-corp
Navigate to Settings tab
Scroll down to Pages section
Under Source, select "Deploy from a branch"
Choose Branch: gh-pages with / (root) folder
Click Save
Step 5: Access Your Site
Your site will be available at: https://bbn7.github.io/zoon-corp/
It may take a few minutes to be live
